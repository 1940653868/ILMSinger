# 歌王ILM —— 基于InternLM-7b的歌词创作大模型
歌王ILM是一个集成了多种语言说唱歌词的大语言模型。该模型使用截止至2023年的音乐平台说唱歌曲歌词作为数据集，基于 InternLM2-Math-7B 模型，通过 xtuner 微调，实现根据不同需求创造歌词的功能。

# 简介

|:--------:| :-------------:|
| InternLM-Chat-7b/InternLM2-Chat-7b | QloRA |
